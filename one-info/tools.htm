Vics One app TechTool 1- Wireless Passwords

Import and Export Wireless Passwords
The command netsh wlan export profile folder="%~dp0\wifi\" key=clear is a Windows batch script instruction used to backup all saved Wi-Fi profiles on a computer to a specific folder, while revealing their passwords in plain text.


SCRIPTS:
    ::Export Wireless Passwords
	MD %~dp0\wifi\
	netsh wlan export profile folder="%~dp0\wifi\" key=clear
	pause

    ::Import Wireless Passwords
	FORFILES /P "%~dp0wifi" /M *.xml /C "cmd /c netsh wlan add profile filename=@path"
	pause

Vics One app TechTool 2- Reset WiFi C*

Setup WIFI Connect to work with your shops wireless Internet Connect to a computer that already has the wireless password setup and set for auto connect, if none available then go ahead and set yours up now. Important to check box to connect automatically when setting up the password, Once setup return to TechTool/Tools/*Reset WIFI C* When pressed two options appear 1-Delete old passwords (Removes old wireless profiles stored) 2-Import Wireless Profiles (Imports the wireless password you just setup with auto connect) Important Note: this creates a wifi-s folder in the same directory as TechTool, this stores the wifi profiles that are used and the folder should be kept with the main program for WIFI Connect to work correctly.

Vics One app TechTool 3- Restores Photo Viewer

To use the lightning-fast classic Windows Photo Viewer on Windows 10 or 11, you must activate it via the registry. Microsoft leaves the underlying program engine (PhotoViewer.dll) hidden inside clean installations. It only associates it with .tif files by default.

Vics One app TechTool 4- Reset Printer Spool

Reset the Windows Printer Spooler, Clear the stuck print queue files from your system storage and restart the spooler service.


Resets Print Spooler 
@ECHO.
@ECHO   ====-----------------------------------------------====
@ECHO             PrintStop2 - Vics TechTool
@ECHO  Stops Printer spool Deletes cashe Restarts Printer Spool
@ECHO   ====-----------------------------------------------====
@ECHO.
@net  stop  spooler
@takeown  /F  %SystemRoot%\System32\Spool\Printers  /R
@icacls  %SystemRoot%\System32\Spool\Printers  /T  /Grant Everyone:F
@c:
@cd %SystemRoot%\system32\spool\printers\
@ECHO.
@ECHO =====================================================
@ECHO             PrintStop2 - Vics TechTool
@ECHO    Deleting cashe - Restarting Printer Spool
@ECHO =====================================================
@ECHO.
@del  /S /Q  %SystemRoot%\System32\Spool\Printers\*.*
@net  start  spooler
pause
